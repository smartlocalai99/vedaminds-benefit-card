# How It Works Card Sizing Design

## Goal

Make the How It Works journey cards visibly larger while preserving the centered member layout, vendor layout, arrows, and mobile stacking.

## Sizes

- Increase card minimum height from 330px to 390px.
- Increase image stage height from 178px to 220px.
- Increase icon tile from 40px to 48px.
- Increase title from 15px to 17px.
- Increase the centered five-card member row maximum width from 1480px to 1650px.
- Move the desktop arrow connector to the vertical center of the larger image stage.

## Verification

- Source assertions verify every approved size.
- ESLint verifies both changed components.
- The Next.js production build verifies compilation.
